# AutoBuy Fabric 1.16.5

Исходники клиентского Fabric-мода для Minecraft `1.16.5`.

## Что умеет

- Открывает меню мода на `M`
- Хранит лимиты цен для предметов
- Сканирует аукцион и сверяет предметы по типу, чаркам, атрибутам, эффектам, lore и NBT
- Автоматически подтверждает подходящие покупки
- Поддерживает профили конфигов
- Поддерживает фильтр моба для спавнера

## Сборка

Требуется Java `8`.

Windows:

```bat
gradlew.bat build
```

Linux/macOS:

```bash
./gradlew build
```

Готовый jar появится в `build/libs`.

## Конфиги

Конфиги мода сохраняются в:

```text
mods/AutoBuy/config
```

## Debug

Для разработчиков в моде есть debug-лог.

Он помогает понять:

- какие предметы видит автобай
- какой предмет он пытается купить
- почему предмет был пропущен
- когда срабатывает обновление аукциона
- когда ломается логика подтверждения покупки

Лог сохраняется в:

```text
mods/debug/debug.txt
```

Если нужно проверить поведение мода, в первую очередь смотри именно этот файл.
