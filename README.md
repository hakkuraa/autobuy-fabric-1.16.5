# AutoBuy Fabric 1.16.5

Исходники клиентского Fabric-мода для Minecraft `1.16.5`.

## Что умеет

- Открывает меню мода на `M`
- Хранит лимиты цен для предметов
- Сканирует аукцион и сверяет предметы по типу, чаркам, атрибутам, эффектам, lore и NBT
- Автоматически подтверждает подходящие покупки
- Поддерживает профили конфигов
- Поддерживает фильтр моба для спавнера

## Сборка

Требуется Java `8`.

Windows:

```bat
gradlew.bat build
```

Linux/macOS:

```bash
./gradlew build
```

Готовый jar появится в `build/libs`.

## Конфиги

Конфиги мода сохраняются в:

```text
mods/AutoBuy/config
```
